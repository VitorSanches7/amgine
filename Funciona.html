COMO FUNCIONA?

Vídeo explicativo:

Passo a Passo para Usar o Emulador da Máquina Enigma:

Passo 1: Familiarizar-se com a Interface
Identifique os componentes: Observe as partes principais do emulador:
Teclado: Onde você digita a mensagem.
Rotores: Seletor de posições e configurações.
Painel de Luzes: Mostra a letra codificada.

Passo 2: Configurar os Rotores
Escolher o número de rotores: Verifique se o emulador permite a seleção do número de rotores (geralmente 3 ou 4).
Definir a posição inicial:
Clique e arraste os rotores para definir suas posições iniciais. Anote essa configuração, pois será necessária para a decodificação.

Passo 3: Inserir a Mensagem
Digite a mensagem: Use o teclado virtual para digitar a mensagem que você deseja codificar.
Pressione a tecla de codificação: Após digitar a mensagem, pressione o botão "Codificar" ou "Encrypt".

Passo 4: Obter a Mensagem Codificada
Leia a mensagem codificada: O painel de luzes exibirá as letras codificadas correspondentes à sua mensagem original.
Anote a mensagem codificada: Guarde essa sequência, pois será usada para a decodificação.

Passo 5: Decodificar a Mensagem
Resetar a configuração: Volte para a tela inicial do emulador e ajuste os rotores para as mesmas posições que você usou para codificar.
Digite a mensagem codificada: Insira a sequência de letras que você obteve.
Pressione a tecla de decodificação: Clique no botão "Decodificar" ou "Decrypt".
Leia a mensagem decodificada: O emulador deve exibir a mensagem original que você digitou.

Passo 6: Experimentar
Testar diferentes mensagens: Tente codificar e decodificar várias mensagens para se familiarizar com o processo.
Alterar a configuração dos rotores: Experimente mudar as posições dos rotores e veja como isso afeta a codificação.
Dicas Finais
Mantenha um registro: Anote suas configurações de rotores e as mensagens codificadas para referência futura.
Leia a documentação: Consulte qualquer guia ou documentação disponível no emulador para entender melhor suas funcionalidades. Divirta-se explorando a máquina Enigma!
